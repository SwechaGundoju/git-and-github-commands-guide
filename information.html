What is Git and Github ?

Git is a distributed version control system that tracks changes in files and enables collaboration on projects.

Github

GitHub is a platform for hosting and collaborating on Git repositories, enabling version control and teamwork for software projects

To check the version of Git

we open the terminal in vs code and just type "git --version"

Git commands

Git commands are primarily related to interacting with Git repositories hosted on GitHub. These commands are used to manage repositories, branches, and collaboration tasks via GitHub.
